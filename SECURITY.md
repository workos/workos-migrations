# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not**
open a public GitHub issue. Instead, report it privately by emailing
[security@workos.com](mailto:security@workos.com).

Please include:

- A description of the issue and its potential impact.
- Steps to reproduce, or a proof-of-concept if possible.
- Any suggested remediation.

We will acknowledge receipt of your report within a few business days and
keep you informed as we investigate and address the issue.

For more information about WorkOS's security practices, see
<https://workos.com/security>.
